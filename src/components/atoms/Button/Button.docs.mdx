import { Button } from './Button';

## Uso básico

```tsx
<Button onClick={() => alert('Hola!')}>Click me</Button>
```

## Props

| Prop    | Tipo         | Descripción                       |
| ------- | ------------ | --------------------------------- |
| variant | string       | 'primary' (default) o 'secondary' |
| ...     | button props | Hereda todas las props de button  |

## Accesibilidad

- Usa `focus:outline-none` y `focus:ring` para accesibilidad.
- Compatible con teclado y lectores de pantalla.
